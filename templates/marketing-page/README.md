# Marketing — starter journey

Copyable marketing-site journey: home → pricing → contact. Copy this whole folder to start your own site; `ds-base.js` has a single `base` line pointing at the design-system root — edit that one line and every page works.

- `MarketingPage.dc.html` — home: hero, product grid, brand band
- `Pricing.dc.html` — tiers, comparison, FAQ
- `Contact.dc.html` — contact form + offices

Showcase twin: `apps/www/registry/kits/website/` covers the same category as plain JSX sections you can lift into a React app. When a section changes in one, update the other.
