# Citation / SourceList

Inline superscript source chips + the source list they resolve to.

`…fares dropped 12%<Citation index={1} href="https://…" title="IATA report" /> this quarter.`
`<SourceList sources={[{ index: 1, title: 'IATA quarterly report', domain: 'iata.org', url: 'https://…' }]} />`

Number citations in reading order; render SourceList once at the end of the message.
