# Docs — starter journey

Copyable documentation journey: quickstart → authentication → errors. Copy this whole folder to start your own docs; `ds-base.js` has a single `base` line pointing at the design-system root — edit that one line and every page works.

- `DocsPage.dc.html` — quickstart with nav tree and table of contents
- `Authentication.dc.html` — guide page with code blocks
- `Errors.dc.html` — reference tables

Showcase twin: `apps/www/registry/kits/docs/` covers the same category as a plain JSX screen you can lift into a React app. When the layout changes in one, update the other.
