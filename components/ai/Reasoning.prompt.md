# Reasoning

Collapsible "thinking" block for model reasoning — pulsing brain glyph while streaming, then "Thought for Ns".

`<Reasoning streaming>Comparing nonstop fares…</Reasoning>`
`<Reasoning duration="4s" defaultOpen={false}>…full chain…</Reasoning>`

Place above the assistant reply inside the message column. Keep collapsed by default when done — reasoning is a receipt, not the answer.
