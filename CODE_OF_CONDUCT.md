# Code of conduct

Meridian is a welcoming project. Participating — issues, PRs, discussions, reviews — means agreeing to this:

## Expected
- Be respectful and direct. Critique work, not people.
- Assume good faith; ask before escalating.
- Welcome newcomers — every maintainer was one.
- Keep discussions on-topic and public where possible.

## Not tolerated
- Harassment, discrimination, or personal attacks of any kind.
- Sexualized language or imagery.
- Publishing others' private information.
- Sustained disruption of discussions or reviews.

## Enforcement
Report violations privately to **conduct@efolusi.com**. Maintainers will review within 7 days and may warn, remove content, or ban — proportionate to the violation. Maintainers who violate this code are held to a higher standard, not a lower one.

This code applies in all project spaces and when representing the project publicly.
